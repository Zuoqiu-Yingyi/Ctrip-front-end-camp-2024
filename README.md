# 旅行日记

## 开发者指南

### 初始化开发环境

1. 本仓库使用 [pnpm 工作空间](https://pnpm.io/workspaces) 与 [Turbo](https://turbo.build/repo) 进行模块管理，因此首先需要全局安装 `pnpm`

    ```shell
    $ npm i pnpm -g
    ```

    更多详情请参考 [Installation | pnpm](https://pnpm.io/installation)
2. 本仓库的 pnpm 工作空间根目录为 `./workspace/`, 因此需要进入 `workspace` 目录

    ```shell
    $ cd workspace
    ```
3. 使用 `pnpm` 初始化工作空间

    ```shell
    $ pnpm i
    ```
